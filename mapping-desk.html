Mapping instructions

Please map the focus area following the instructions below. Follow the OSM Wiki for anything not covered here.

Roads

"Highway" tag and its values:

  1. Type: highway=. See also here (might need editing) and here (Saudi still missing).
  2. Name: Tags name= and name:en=.
  3. Oneway?: oneway=yes, where applicable. For devided roads please use dual_carriageway=yes.
  4. Speed limit: Use maxspeed=.
  5. Number and type of lanes: Use the tags lanes= or :lanes.
  6. Reference number: ref=
  7. Nice-to-have tags:
    • surface=
    • sidewalk=left/right/both/no
    • parking=left/right/both, Shoulder lanes

Bridges and tunnels:

  1. Bridges: bridge= and layer=.
  2. Tunnels: tunnel= and layer=.

Roundabouts and other junctions:

  1. junction=

Traffic signs and lights

Traffic signs:

  1. Stop sign: highway=stop.
  2. Other signs: traffic_sign=. Most interesting signs: maxwidth, maxheight, maxweight.

Traffic lights:

  1. highway=traffic_signals.

Destination signs:

  1. destination=. Use also in combination with Lanes!

Other map features

List of features we would like you to map beyond highways and traffic signs/lights. Please note that that list might be incomplete.

  1. Road bumps: traffic_calming=.
  2. Speed cameras: Relation:enforcement. Please note that this is not a simple tag but a "relation" (a set of tags). Please make sure that you understand relations before mapping.

Please tag every changeset with source=survey;image;Hudhud to indicate the source of the information.
